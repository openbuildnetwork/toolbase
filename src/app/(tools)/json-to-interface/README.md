# JSON to Interface Generator

## Overview

Boost your productivity by instantly converting JSON data into type-safe code for your favorite programming languages. This tool parses your JSON and automatically generates specific structs, interfaces, or classes, saving you the tedious work of manual typing.

## Features

- **Multi-Language Support**:
  - TypeScript (Interfaces/Types)
  - Go (Structs)
  - Python (Pydantic/Dataclasses)
  - Java (Classes)
  - Rust (Structs)
  - Swift, Kotlin, C#, and more.
- **Instant Preview**: Paste JSON and see the generated code immediately.
- **Customizable**: simple and intuitive interface.

## Privacy Guarantee

- **100% Local**: Parsing and code generation happen entirely within your browser.
- **Secure**: Your API responses and data structures are never shared.

## Usage

1. Paste your JSON object into the left panel.
2. Select your target language from the dropdown.
3. Copy the generated code from the right panel.

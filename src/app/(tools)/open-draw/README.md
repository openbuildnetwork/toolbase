# OpenDraw

A premium, infinite-canvas diagramming tool.

## Purpose

Create flows, diagrams, and charts with a simple drag-and-drop interface.

## Inputs

- Drag and drop shapes from the sidebar
- Local .json files for loading diagrams

## Outputs

- Visual diagrams
- .json files saved locally to your device

## Privacy Guarantee

All processing is client-side. Files are saved directly to your disk using the File System Access API. No data is sent to any server.
